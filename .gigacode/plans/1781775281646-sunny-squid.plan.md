# План переписи приложения To-Do List

## Описание проекта
Простое веб-приложение для управления задачами (To-Do list) без авторизации с сохранением данных в SQLite. Добавлены две связанные модели: Проекты и Задачи.

## Требования
- Laravel 11 без Laravel Breeze
- PHP 8.2+
- SQLite для хранения данных
- Две модели: Project и Task (один проектMany задач)
- Tailwind CSS для стилизации

## Реализованные файлы

### Модели
- `app/Models/Project.php` - модель проекта с связью many-to-many к задачам
- `app/Models/Task.php` - модель задачи с связью many-to-one к проекту

### Миграции
- `database/migrations/2026_06_18_094724_create_projects_table.php` - создание таблицы проектов
- `database/migrations/2026_06_18_094728_create_tasks_table.php` - создание таблицы задач

### Контроллеры
- `app/Http/Controllers/ProjectController.php` - CRUD операции для проектов
- `app/Http/Controllers/TaskController.php` - CRUD операции для задач

### Маршруты (routes/web.php)
- `GET /projects` - список проектов
- `POST /projects` - создание проекта
- `GET /projects/create` - форма создания проекта
- `GET /projects/{project}` - задачи конкретного проекта
- `POST /projects/{project}/tasks` - создание задачи в проекте
- `PATCH /projects/{project}/tasks/{task}` - обновление статуса задачи
- `DELETE /projects/{project}/tasks/{task}` - удаление задачи
- `DELETE /projects/{project}` - удаление проекта

### Представления (resources/views/)
- `projects/index.blade.php` - список проектов
- `projects/create.blade.php` - форма создания проекта
- `projects/show.blade.php` - задачи проекта с формой добавления
- `layouts/app.blade.php` - основной шаблон приложения
- `layouts/navigation.blade.php` - упрощенная навигация без авторизации
- `components/app-layout.blade.php` - компонент основного макета
- `components/input-error.blade.php` - компонент ошибок валидации
- `components/input-label.blade.php` - компонент меток
- `components/text-input.blade.php` - компонент текстового поля
- `components/dropdown.blade.php` - компонент выпадающего меню
- `components/application-logo.blade.php` - упрощенный логотип

### Конфигурация
- `.env` - настроена SQLite и файловая сессия
- `database/todo.sqlite` - база данных SQLite

## Функционал
- Создание проектов с названием и описанием
- Добавление задач в проекты
- Отметка задач как выполненные (checkbox)
- Удаление задач и проектов
- Отображение статистики (всего задач, невыполненных)

## Проверка
1. Запустить сервер: `php artisan serve`
2. Перейти на `http://localhost:8000/projects`
3. Протестировать:
   - Создание нового проекта
   - Добавление задач в проект
   - Отметка задач как выполненных
   - Удаление задач и проектов
   - Проверка статистики задач
